# 会话生命周期

## 概览

```
创建 Session → 订阅 → 发送消息 → 接收流式响应 → 清理
```

---

## 1. 创建会话

### TypeScript

```typescript
const { session_id } = await client.sessions.createSession("分析这份数据")
```

### Python

```python
session = await client.create_session("分析这份数据")
session_id = session.id
```

### REST

```http
POST /api/sessions
Content-Type: application/json

{
  "intent": "分析这份数据",
  "solution_id": "data-analysis",
  "model": "claude-sonnet-4-20250514"
}
```

---

## 2. 订阅会话

通过 Socket.IO 订阅实时更新：

```typescript
const socket = client.socket()
socket.connect()
socket.emit("session:subscribe", { session_id: sessionId })
```

订阅后即可接收该会话的所有 `turn:*`、`chat:*`、`session:*` 事件。

---

## 3. 发送消息

```typescript
socket.emit("chat:send", {
  session_id: sessionId,
  message: "请帮我统计每月销售额"
})
```

### 带附件

```typescript
socket.emit("chat:send", {
  session_id: sessionId,
  message: [
    { type: "text", text: "分析这张图片" },
    { type: "image_url", image_url: { url: "data:image/png;base64,..." } }
  ]
})
```

### 回答 AskUserQuestion

当工具状态为 `awaiting_answer` 时：

```typescript
socket.emit("chat:send", {
  session_id: sessionId,
  message: "用户确认",
  askuser_answer: {
    tool_call_id: "tc_xxx",
    selections: { "0": [1] },     // 第 0 题选第 1 项
    custom: {}
  }
})
```

---

## 4. 接收流式响应

事件顺序：

```
chat:start
  └→ turn:start  (user turn)
  └→ turn:end    (user turn)
  └→ turn:start  (assistant turn, status: streaming)
       └→ turn:patch (add_content: text delta)
       └→ turn:patch (add_content: thinking)
       └→ turn:patch (add_tool_call)
       └→ turn:patch (tool_result)
       └→ turn:patch (add_content: text delta)
  └→ turn:end    (assistant turn, status: completed)
  └→ [可能有更多 turn 循环...]
chat:end
```

> 一次 `chat:send` 可能触发多轮 turn（agent 循环调用工具）。

---

## 5. 停止生成

```typescript
socket.emit("chat:stop", { session_id: sessionId })
```

### Python

```python
await client.stop(session_id)
```

---

## 6. 读取历史

`chat:end` 后可拉取完整历史：

```typescript
// 渲染后的 Turn 投影（推荐）
const turns = await client.sessions.getSessionTurns(sessionId)

// 原始历史树
const history = await client.sessions.getSessionHistory(sessionId)
```

---

## 7. 清理

```typescript
// 取消订阅
socket.emit("session:unsubscribe", { session_id: sessionId })

// 断开连接（Node.js 必须，否则进程挂起）
socket.disconnect()
```

### 删除会话

```typescript
await client.sessions.deleteSession(sessionId)
```

## 会话模式

| 模式 | 说明 |
|------|------|
| `planning` | 智能体先制定结构化计划，不直接执行工具 |
| `executing` | 智能体直接执行动作和工具 |

切换模式：

```typescript
// REST
await fetch(`/api/sessions/${sessionId}/mode`, {
  method: "POST",
  body: JSON.stringify({ mode: "planning" })
})
```

## What-If 重跑

从某一步重新运行，使用修改后的输入：

```typescript
socket.emit("chat:send", {
  session_id: sessionId,
  message: "改用不同的方法",
  whatif: {
    from_step: 3,
    quotes: [],
    deprecate_entry_ids: ["entry_1", "entry_2"]
  }
})
```

- `from_step`：保留前 N 步，从第 N+1 步重跑
- `deprecate_entry_ids`：标记为废弃的条目（UI 可过滤）

---

## 上下文压缩（Compaction）

当对话超出上下文窗口时自动触发，也可手动触发：

```typescript
socket.emit("chat:compact", { session_id: sessionId })
```

压缩后 Turn 的 `kind` 为 `"compaction"`，包含摘要信息和压缩比。

## 完整示例：TypeScript

```typescript
import { BladeClient } from "@blade-hq/agent-kit/client"

const client = new BladeClient({
  baseUrl: "https://blade.example.com",
  token: "sk-blade-v2-..."
})

// 1. 创建会话
const { session_id } = await client.sessions.createSession("帮我分析数据")

// 2. 连接 & 订阅
const socket = client.socket()
socket.connect()
socket.emit("session:subscribe", { session_id })

// 3. 监听事件
socket.on("turn:patch", (patch) => {
  if (patch.patch_type === "add_content" && patch.data.text_delta) {
    process.stdout.write(patch.data.text_delta)
  }
})

socket.on("chat:end", async () => {
  const turns = await client.sessions.getSessionTurns(session_id)
  console.log("Done!", turns.length, "turns")

  // 4. 清理
  socket.emit("session:unsubscribe", { session_id })
  socket.disconnect()
})

// 5. 发送消息
socket.emit("chat:send", { session_id, message: "统计每月销售额" })
```

## 完整示例：Python

```python
from blade_agent_kit import BladeAgentClient

async with BladeAgentClient(
    "http://127.0.0.1:8020",
    token="sk-blade-v2-...",
) as client:
    # 1. 创建会话
    session = await client.create_session("帮我分析数据")

    # 2. 流式聊天
    async for event in client.chat(session.id, "统计每月销售额"):
        if event.kind == "turn:end":
            print(f"Turn completed: {event.data}")

    # 3. 读取历史
    history = await client.get_history(session.id)
```

# 核心类型

---

## TurnProjection

Turn 的完整快照，代表一轮对话交互。

```typescript
interface TurnProjection {
  id: string
  sequence: number
  turn_id: string
  loop_id: string
  kind: "message" | "compaction"
  role: "user" | "assistant" | "system"
  status: "streaming" | "completed" | "paused" | "failed" | "interrupted"
  blocks: ContentBlock[]
  tool_calls: ToolCallProjection[]
  model: string | null
  usage: Record<string, unknown> | null   // { input_tokens, output_tokens, ... }
  duration_ms: number
  started_at: string | null
  context_window: number
  memory_refs: MemoryRef[] | null

  // 子智能体（Fork）
  parent_fork_tool_call_id: string | null

  // Compaction（上下文压缩）
  compaction_id: string | null
  summary_preview: string | null
  summary_full: string | null
  archived_count: number | null
  tokens_before: number | null
  tokens_after: number | null
  saved_ratio: number | null              // 0~1
  trigger: "auto" | "manual" | "forced_retry" | null
  failure_reason: string | null
}
```

## ContentBlock

Turn 内的内容元素。

```typescript
interface ContentBlock {
  type: ContentBlockType
  content: unknown               // 类型随 type 变化
  tool_call_id?: string | null
  tool_name?: string | null
  display_name?: string | null   // 工具友好名称
}

type ContentBlockType =
  | "text"                   // 文本内容
  | "thinking"               // 思考过程
  | "tool_use"               // 工具调用声明
  | "tool_result"            // 工具执行结果
  | "tool_ui"                // 工具 UI 卡片
  | "tool_bridge"            // 工具桥接动作
  | "system_notification"    // 系统通知
  | "mode_change"            // 模式切换
  | "planning_enter"         // 进入 Planning 模式
  | "planning_exit"          // 退出 Planning 模式
  | "plan_status"            // 计划状态
  | "ask_user_answer"        // 用户回答
```

## ToolCallProjection

工具调用详情。

```typescript
interface ToolCallProjection {
  id: string
  tool_name: string
  display_name: string
  arguments: string           // JSON 字符串，需 JSON.parse()
  status: ToolCallStatus
  result: unknown | null
  duration_ms: number | null
  pending_question_ref: PendingQuestionRef | null
}

type ToolCallStatus =
  | "pending"            // 执行中
  | "done"               // 完成
  | "error"              // 出错
  | "cancelled"          // 已取消
  | "awaiting_answer"    // 等待用户回答

interface PendingQuestionRef {
  child_loop_name: string
  child_tool_call_id: string
  description?: string | null
}
```

## PatchEnvelope

增量更新信封（用于 `turn:patch` 事件）。

```typescript
interface PatchEnvelope {
  sequence: number
  turn_id: string
  loop_id: string
  patch_type: string        // "add_content" | "add_tool_call" | "set_status" | ...
  data: Record<string, unknown>
}
```

## ChatMessage（React Headless）

`useChat()` 返回的消息格式。

```typescript
interface ChatMessage {
  role: "user" | "assistant" | "tool" | "error"
  content: MessageContent
  reasoning?: string                // 思考过程（可折叠）
  tool_calls?: ToolCallInfo[]
  status?: "streaming" | "completed" | "paused" | "failed" | "interrupted"
  duration_ms?: number
  entry_id?: string                 // 用作 React key
  kind?: string                     // "planning_enter" / "planning_exit" 等
  memory_refs?: MemoryRefInfo[]
  blocks?: ContentBlock[]           // block 级渲染
  compaction?: CompactionInfo
}

type MessageContent = string | MessageContentPart[]

type MessageContentPart =
  | { type: "text"; text: string }
  | { type: "image_url"; image_url: { url: string } }
  | { type: "file"; name: string; data: string }
```

## Runtime 类型（Python Core）

### ToolSpec

工具定义。

```python
@dataclass(frozen=True)
class ToolSpec:
    name: str               # 全局唯一运行时名称
    display_name: str
    description: str
    parameters: dict[str, Any]
```

### ToolResult

工具执行输出。

```python
@dataclass
class ToolResult:
    output: str
    is_error: bool = False
    metadata: dict[str, Any] | None = None
    pause: dict[str, Any] | None = None         # 暂停信号（AskUser 等）
    extra_content: list[dict[str, Any]] | None = None
```

### ToolCallContext

工具调用上下文。

```python
@dataclass(frozen=True)
class ToolCallContext:
    session_id: str
    loop_name: str
    cwd: str
    tool_call_id: str | None = None
    max_turns: int | None = None
    mode: str = "executing"       # "planning" | "executing"
    # ... 及回调函数
```

### AgentLoopResult

Agent 循环结果。

```python
@dataclass(frozen=True)
class AgentLoopResult:
    output: str
    turns_used: int
    pause_tool: dict[str, Any] | None = None
    finish_reason: str = "completed"   # "completed" | "max_turns" | "tool_error"
    terminal_payload: dict[str, Any] | None = None
```

### LLMRequestConfig

LLM 请求配置。

```python
@dataclass(frozen=True)
class LLMRequestConfig:
    model: str | None = None
    enable_thinking: bool | None = None
    extra_headers: dict[str, str] | None = None
    context_window: int = 0
```

## Tool _meta 扩展

工具结果可通过 `_meta` 字段向宿主页面传递动作和 UI 卡片。

### Bridge（页面动作）

```json
{
  "ok": true,
  "_meta": {
    "bridge": {
      "action": "map.highlight",
      "payload": { "cityIds": ["hangzhou", "ningbo"] }
    }
  }
}
```

### UI Card（内联/预览卡片）

```json
{
  "_meta": {
    "ui": {
      "resourceHTML": "<!doctype html>...",
      "resourceUri": "https://...",
      "target": "inline | preview",
      "height": 320,
      "title": "卡片标题"
    }
  }
}
```

- `resourceHTML`：完整 HTML，通过 iframe `srcdoc` 渲染
- `resourceUri`：外部 URL，通过 iframe `src` 渲染
- `target`：`inline`（聊天区内联）或 `preview`（右侧预览面板）

# 云电脑

## 什么是云电脑

云电脑是一个持久化的沙箱环境，类似一台专属虚拟机。与普通对话中的临时环境不同，云电脑中的文件和安装的软件在会话结束后依然保留，下次打开时一切都还在。

## 我的文档

在云电脑模式下，左侧栏会显示用户文件树，方便浏览和管理文件。所有文件持久化保存，不会因会话结束而丢失。

```
┌──────────────────┬──────────────────────────┐
│  我的文档          │                          │
│  ├── 报表/         │                          │
│  │   ├── Q1.xlsx   │     对话区域              │
│  │   └── Q2.xlsx   │                          │
│  ├── 合同/         │                          │
│  │   └── 模板.docx │                          │
│  └── 笔记.md       │                          │
│                    │                          │
│  <-- 文件树面板     │                          │
└──────────────────┴──────────────────────────┘
```

## 升级系统

云电脑提供一键升级功能，可以将系统中的软件包和工具更新到最新版本，确保运行环境保持最新。

## 重置云电脑

如果环境出现异常或需要重新开始，可以在高级设置中重置云电脑：

```
┌──────────────────────────────────┐
│  高级设置                         │
│                                  │
│  云电脑管理                       │
│  ┌────────────────────────────┐  │
│  │  [ 重置云电脑 ]              │  │  <-- 重置按钮
│  │  恢复为初始状态               │  │
│  │  已保存的用户文件不受影响      │  │
│  └────────────────────────────┘  │
└──────────────────────────────────┘
```

重置后环境恢复为初始状态，但「我的文档」中的用户文件不受影响。

## 工作区与文件隔离

每个用户拥有独立的云电脑工作区，文件和环境互不影响。你在云电脑中的所有操作不会影响其他用户，其他用户也无法访问你的文件。

# 对话与交互

## 发送消息

在对话框中输入文字，按回车或点击发送按钮即可。智能体会根据当前解决方案和角色，调用相关技能来处理你的请求。

## 语音输入

点击输入框旁的麦克风按钮开始录音，系统会实时将语音转为文字（支持豆包 Seed-ASR、Qwen3-ASR）。录音过程中会显示波形动画，完成后文字自动填入输入框。

## 上传文件

对话时可以上传文件作为参考材料，支持的格式包括 Markdown、纯文本、CSV、图片等。

两种上传方式：
- 点击对话框旁的上传按钮选择文件
- 直接将文件拖拽到对话框

上传时会显示进度条，完成后智能体可以读取文件内容并据此回答。

## 图片附件

消息中可以直接粘贴或上传图片，智能体支持多模态理解，能够识别和分析图片中的内容。

## 规划模式与执行模式

智能体有两种工作模式：

- **规划模式**：智能体只输出方案，不执行任何操作。适合你想先看看方案再决定是否动手的场景。
- **执行模式**：智能体直接动手完成任务（写代码、操作文件等）。适合你已经明确需求、希望直接出结果的场景。

你可以在对话界面中随时切换模式。

## 查看智能体执行过程

智能体工作时，你可以实时查看执行细节：

- **流式输出**：智能体的回复逐字出现，无需等到全部完成。
- **深度思考**：遇到复杂问题时，智能体会展示思考过程。思考完成后自动折叠，不影响阅读。
- **工具调用**：智能体调用工具时会显示调用详情，包括工具名、传入参数和返回结果。
- **子智能体**：复杂任务会拆分给子智能体处理，每个子任务独立显示进度。子智能体可能会暂停并向你提问（AskUserQuestion），回答后继续执行。
- **上下文压缩**：长对话时系统自动压缩历史上下文，保持智能体的理解能力。

```
+------------------------------------------------------------------+
|  对话界面                                                         |
|------------------------------------------------------------------|
|                                                                  |
|  [用户消息]  你好，帮我分析这份报告                                 |
|                                                                  |
|  [智能体]                                                         |
|    > 深度思考 (已折叠，点击展开)                                    |
|                                                                  |
|    我来分析这份报告的内容...                                       |
|                                                                  |
|    > 工具调用: read_file("report.csv")        ← 点击展开查看详情   |
|    > 子智能体: 数据分析                       ← 独立显示进度       |
|      > AskUserQuestion: 请确认分析维度         ← 等待你回答        |
|                                                                  |
+------------------------------------------------------------------+
| [文本输入框]  [麦克风] [上传] [规划/执行切换]     [发送]            |
+------------------------------------------------------------------+
```

## 预览功能

智能体生成的产物可以直接在对话中预览：

- **代码产物预览**：生成的代码可以直接运行和预览
- **文件预览**：文档、表格等文件内容直接展示
- **iframe 预览**：网页类产物通过内嵌窗口预览
- **PDF 和图片**：直接在对话中查看

## 模型选择

对话中可以动态切换 LLM 模型。管理员可在后台配置可用模型的白名单。

## 常见失败与处理

| 现象 | 建议操作 |
|------|----------|
| 智能体无响应 | 等待片刻后刷新页面重试 |
| 回复中断 | 尝试缩小问题范围，重新提问 |

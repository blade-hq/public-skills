# 和智能体对话

## 发送消息

在对话框中输入文字，按回车或点击发送按钮即可。智能体会根据当前解决方案和角色，调用相关技能来处理你的请求。

## 上传文件

对话时可以上传文件作为参考材料。支持的格式包括：

- Markdown 文档
- 纯文本文件
- CSV 表格
- 其他常见文档格式

点击对话框旁的上传按钮，选择文件即可。上传后，智能体可以读取文件内容并据此回答。

## 规划模式与执行模式

智能体有两种工作模式：

- **规划模式**：智能体只输出方案，不执行任何操作。适合你想先看看方案再决定是否执行的场景。
- **执行模式**：智能体直接动手完成任务。适合你已经明确需求、希望直接出结果的场景。

你可以在对话界面中切换模式。

## 查看执行过程

智能体工作时，你可以实时查看执行细节：

- **流式输出**：智能体的回答会逐步显示，无需等到全部完成
- **工具调用详情**：展开可查看智能体调用了哪些工具、传入了什么参数、返回了什么结果
- **子智能体进度**：如果主智能体调度了子智能体，你可以看到每个子智能体的工作状态

## 常见问题与处理

| 现象 | 可能原因 | 建议操作 |
| --- | --- | --- |
| 无响应 | 服务繁忙或网络异常 | 等待片刻后刷新页面重试 |
| 回答中断 | 输出内容过长或超时 | 尝试缩小问题范围，重新提问 |
| 文件读不到 | 文件格式不支持或上传失败 | 检查文件格式，重新上传 |
| 结果不符预期 | 问题描述不够清晰 | 补充细节、拆分问题，或切换到更匹配的角色 |

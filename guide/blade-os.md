# Blade OS 桌面

Blade OS 是一个运行在浏览器中的桌面环境（`http://<host>`，端口 80），模拟了操作系统的交互方式。你可以像使用电脑桌面一样，打开应用、管理窗口、操作文件。

## 桌面布局

```
+------------------------------------------------------------------+
|                                                                  |
|   +----------+  +----------+  +----------+                      |
|   | 智能助手  |  | 软件工厂  |  | 能力中心  |  ...  ← 应用图标区域 |
|   +----------+  +----------+  +----------+                      |
|                                                                  |
|   +----------------------------------------------------------+  |
|   | 窗口标题栏                          [最小化][最大化][关闭] |  |
|   |----------------------------------------------------------|  |
|   |                                                          |  |
|   |                    窗口内容区域                            |  |
|   |              (应用在这里运行和展示)                        |  |
|   |                                                          |  |
|   +----------------------------------------------------------+  |
|                                                                  |
+------------------------------------------------------------------+
| [应用1] [应用2] [应用3]                            ← 底部任务栏   |
+------------------------------------------------------------------+
```

- **应用图标区域**：展示所有可用的应用入口，点击打开对应应用
- **窗口区域**：应用以窗口形式展示，支持多窗口同时显示
- **底部任务栏**：显示已打开的应用，方便快速切换

## 窗口操作

- **打开**：点击应用图标打开窗口
- **关闭**：点击窗口右上角的关闭按钮
- **拖拽**：按住窗口标题栏拖动，调整窗口位置
- **最大化**：双击标题栏或点击最大化按钮，窗口铺满整个屏幕

## 内置应用

| 应用 | 地址 | 说明 |
|------|------|------|
| 智能助手 | `http://<host>:8020/` | 智能体对话入口 |
| 软件工厂 | `http://<host>:8020/studio/vibe-coding` | 软件工程解决方案 |
| 技能开发 | `http://<host>:8020/studio/skill-editor` | 技能编辑与调试 |
| 能力中心 | `http://<host>:8010/` | 管理智能体的能力与配置 |
| 帮助文档 | 桌面内置组件 | 平台使用说明 |
| 文件管理器 | 桌面内置组件 | 浏览和管理文件 |
| 高级设置 | 桌面内置组件 | 系统与用户配置（含 LLM 观测） |
| 文件预览 | 桌面内置组件 | 预览文档、图片等文件 |

## 文件管理

通过「文件管理器」应用，你可以：

- **上传文件**：将本地文件上传到平台
- **下载文件**：将平台上的文件下载到本地
- **预览文件**：直接在浏览器中预览文档、图片等，无需下载

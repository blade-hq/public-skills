# Blade OS 桌面

Blade OS 是一个运行在浏览器中的桌面环境，模拟了操作系统的交互方式。你可以像使用电脑桌面一样，打开应用、管理窗口、操作文件。

## 桌面布局

- **任务栏**：显示已打开的应用，方便快速切换
- **窗口区域**：应用以窗口形式展示，支持多窗口同时显示
- **应用列表**：展示所有可用的应用入口

## 窗口操作

- **打开**：点击应用图标打开窗口
- **关闭**：点击窗口右上角的关闭按钮
- **拖拽**：按住窗口标题栏拖动，调整窗口位置
- **最大化**：双击标题栏或点击最大化按钮，窗口铺满整个屏幕

## 内置应用

| 应用 | 说明 |
| --- | --- |
| 智能助手 | 智能体对话入口，发起和管理对话 |
| 软件工厂 | 软件工程解决方案，辅助开发流程 |
| 技能开发 | 编辑和调试智能体技能 |
| 能力中心 | 管理智能体的能力与配置 |
| 帮助文档 | 平台使用说明和指引 |
| 文件管理器 | 浏览和管理文件 |
| 高级设置 | 系统与用户配置 |
| 文件预览 | 预览文档、图片等文件 |

## 文件管理

通过「文件管理器」应用，你可以：

- **上传文件**：将本地文件上传到平台
- **下载文件**：将平台上的文件下载到本地
- **预览文件**：直接在浏览器中预览文档、图片等，无需下载

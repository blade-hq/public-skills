# 会话管理

## 会话列表

左侧边栏展示所有历史会话，按时间倒序排列。点击任意会话即可切回该对话继续交流。

```
┌─────────────────────────────────┐
│  🔍 搜索会话...                  │  <-- 搜索框
├─────────────────────────────────┤
│  📌 财务报表分析        3小时前   │  <-- 置顶会话（📌 图标）
│  📌 合同条款审查        昨天      │
├─────────────────────────────────┤
│  产品需求整理           今天      │
│  周报生成              昨天      │
│  数据清洗任务           3天前     │
├─────────────────────────────────┤
│  [ 导入 ]  [ 导出 ]             │  <-- 导入/导出按钮
└─────────────────────────────────┘
```

## 置顶与搜索

- **置顶**：右键点击会话或使用菜单，可将重要会话置顶。置顶会话始终显示在列表顶部，方便快速访问。
- **搜索**：在列表顶部的搜索框中输入关键词，快速定位目标会话。

## 导入与导出

- **导出**：将单个会话导出为文件，方便备份或分享给他人。
- **导入**：从导出的文件中恢复会话，支持单个导入和批量导入。

## 删除会话

在会话列表中，右键点击或使用菜单选择删除，即可移除不再需要的会话。删除后不可恢复，请谨慎操作。

## 自动标题

新建会话后，智能体会根据对话内容自动生成一个简短标题，无需手动命名。

# 定时任务

## 什么是定时任务

定时任务允许你设置智能体在指定时间或按周期自动执行任务。适用于需要定期重复的工作，例如：

- 每天早上生成销售日报
- 每周五下午汇总本周数据
- 每月初检查合同到期情况

## 创建定时任务

有两种方式创建定时任务：

1. **侧边栏入口**：在左侧边栏找到「定时任务」入口，点击创建。
2. **对话中创建**：直接在对话中告诉智能体，例如"每天早上 9 点帮我生成销售日报"，智能体会自动创建定时任务。

## 管理定时任务

在定时任务列表中，可以查看所有已创建的任务：

```
┌───────────────────────────────────────────────┐
│  定时任务                    [ + 创建任务 ]     │  <-- 创建入口
├────┬──────────────┬──────────┬────────────────┤
│ 编号│ 任务名称      │ 周期      │ 状态           │
├────┼──────────────┼──────────┼────────────────┤
│ 1  │ 销售日报生成   │ 每天 9:00 │ ✅ 已启用       │  <-- 任务状态
│ 2  │ 周数据汇总    │ 每周五    │ ✅ 已启用       │
│ 3  │ 合同到期检查   │ 每月 1 日 │ ⏸ 已禁用       │
└────┴──────────────┴──────────┴────────────────┘
```

支持的操作：

- **启用 / 禁用**：暂停或恢复任务，不影响任务配置。
- **删除**：彻底移除任务。

## 查看执行结果

每次定时任务执行后，系统会记录执行状态和输出内容。点击任务可查看历史执行记录，了解每次执行是否成功以及具体的输出结果。

# 快速开始

## 获取账号

### 公网演示环境

联系销售团队申请试用账号。

### 私有化部署

有两种方式创建账号：

- **欢迎向导**：首次部署后，第一个登录的用户会进入欢迎向导，按提示创建管理员账号。
- **管理员创建**：管理员登录 Blade OAuth 管理后台（`http://<host>:19000/admin`），为团队成员创建账号。

## 登录平台

打开浏览器，访问 `http://<host>`（Blade OS 桌面，端口 80），使用账号密码登录。

## 进入桌面

登录成功后，自动进入 Blade OS 浏览器桌面。桌面是你使用所有功能的起点。

![Blade OS 桌面](/images/blade-os-desktop.png)

## 发起第一次对话

1. 在桌面上打开「智能助手」应用（也可以直接访问 `http://<host>:8020`）
2. 在对话框中输入你的问题
3. 发送消息

智能体收到问题后会自动开始工作：查询内置数据库，深入分析数据，连续执行多个工具调用，最终给出有依据的结论。整个过程你可以实时看到智能体的思考和操作。

# 记忆功能

## 什么是记忆

智能体可以记住跨会话的关键信息，例如你的偏好、项目背景、常用术语等。记忆一旦创建，智能体会在后续所有对话中自动参考这些信息，无需你反复说明。

## 查看记忆列表

在页面右上角的用户菜单中，可以查看智能体已经记住的所有内容。

```
┌──────────────────────────────────────┐
│  用户头像 ▼                          │  <-- 点击展开用户菜单
│  ┌────────────────────────────────┐  │
│  │  个人设置                       │  │
│  │  记忆管理    <-- 入口           │  │
│  │  退出登录                       │  │
│  └────────────────────────────────┘  │
└──────────────────────────────────────┘

┌──────────────────────────────────────┐
│  记忆管理                            │
├──────────────────────────────────────┤
│  ● 用户偏好：报表格式用 PDF    [ 删除 ] │  <-- 记忆条目
│  ● 项目背景：当前在做 Q2 预算   [ 删除 ] │
│  ● 常用数据源：用友 ERP 系统   [ 删除 ] │
└──────────────────────────────────────┘
```

## 手动创建记忆

直接在对话中告诉智能体需要记住的信息即可。例如：

- "请记住：我的报表统一用 PDF 格式"
- "记住这个项目的截止日期是 6 月底"

智能体会自动将这些信息存入记忆。

## 删除记忆

在记忆列表中，点击条目右侧的删除按钮即可移除该条记忆。删除后，智能体不再参考该信息。
